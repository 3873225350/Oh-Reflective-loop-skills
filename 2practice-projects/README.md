# Practice Projects

Hands-on projects built with the reflective loop skills in this repository.

## Projects

- `portfolio-resume/` — an interactive personal portfolio and resume site for showcasing repositories, systems, and research projects.
- `live-loop-cinema/` — an interactive HTML-PPT deck for turning reflective-loop ideas into click-through video/storytelling scenes.
- `reflective-loop-simulator/` — a hands-on loop mechanism simulator inspired by the Claude Code unpacking topics, with selectable runtimes and event playback.
- `oh-my-icon/` — a dependency-free SVG animation CLI and skill draft for turning iconfont-style SVGs into animated HTML/SVG assets.
