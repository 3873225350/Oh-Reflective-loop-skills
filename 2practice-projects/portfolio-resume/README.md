# Portfolio Resume

An interactive single-page portfolio built as the first practice project for the reflective loop skills.

## Open

Open `index.html` directly in a browser.

## Edit Content

Most portfolio content lives in `app.js`:

- `profile`
- `repositories`
- `timeline`
- `signals`

The page is intentionally static and dependency-free, so it can be hosted on GitHub Pages or any static file server.
