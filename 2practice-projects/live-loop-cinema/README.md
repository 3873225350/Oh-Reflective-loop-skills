# Live Loop Cinema

An interactive HTML-PPT project inspired by the local LivePPT reference in:

`/home/clashuser/hzh/item_bo/learn-likecc/reference/reference_design_ui/LivePPT`

The goal is to present Reflective Loop Skills as a cinematic, clickable, online talk instead of a static slide deck.

## Open

Open `index.html` directly in a browser.

## Controls

- Arrow keys: previous / next scene
- Space: play / pause autoplay
- Number keys 1-6: jump to a scene
- Mouse movement: controls the live visual field

## Content Direction

This deck is meant for product videos, online demos, and short conference-style explanation clips. Each scene has one claim:

1. Long tasks fail because context does not persist into action.
2. Roadmaps become reusable slow state.
3. Active tasks become fast adapters.
4. Checks become loss signals.
5. Failure banks become memory.
6. MOA turns independent loop skills into a coordinated graph.
